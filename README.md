# BEUShareBox

Kullanıcıların keşfettikleri ürünleri paylaşabildiği, beğenebildiği ve yorum yapabildiği mini bir sosyal medya web uygulamasıdır. Herhangi bir kütüphane veya framework kullanılmadan, tamamen Vanilla JS ile Single Page Application (SPA) olarak geliştirilmiştir.

## Özellikler
* **Temel İşlevler:** Ürün ekleme, silme, beğenme ve yorum yapma.
* **Filtreleme ve Arama:** Kategoriye göre listeleme ve anlık ürün arama.
* **Veri Kalıcılığı:** LocalStorage entegrasyonu ile verilerin tarayıcıda saklanması.
* **Tasarım:** CSS Grid ve Flexbox ile tamamen responsive (mobil uyumlu) modern arayüz.
* *(Eklediğin ileri düzey özellikler varsa buraya yazabilirsin: Dark Mode, Sürükle-Bırak, Görsel Yükleme vb.)*

## Teknolojiler
* HTML5 (Semantik)
* CSS3 (Özel Değişkenler, Animasyonlar)
* Vanilla JavaScript (ES6+, DOM Manipülasyonu)
